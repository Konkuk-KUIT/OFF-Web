# OFF Frontend

AI 기반 1인 크리에이터 및 프로젝트 제작자를 위한  
올인원 프로젝트 플랫폼 **OFF**의 프론트엔드 레포지토리입니다.

---

## 📌 프로젝트 개요

OFF는 프로젝트 기획부터  
AI 자동 견적, 파트너 매칭, 협업, 관리까지  
모든 과정을 하나로 연결하는 플랫폼입니다.

프론트엔드는  
**사용자가 복잡함 없이 프로젝트를 생성·관리하고  
파트너와 즉시 협업할 수 있는 경험**에 집중합니다.
